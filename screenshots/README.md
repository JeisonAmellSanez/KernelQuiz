# Screenshots

Esta carpeta contiene las capturas de pantalla de la aplicación KernelQuiz.

## Instrucciones

Para agregar capturas de pantalla a la documentación:

1. Toma capturas de pantalla desde tu dispositivo Android o emulador
2. Guárdalas en esta carpeta con nombres descriptivos:
   - `01-login.png` - Pantalla de inicio de sesión
   - `02-register.png` - Pantalla de registro
   - `03-welcome.png` - Pantalla de bienvenida
   - `04-categories.png` - Selección de categorías
   - `05-quiz.png` - Interfaz del quiz
   - `06-results.png` - Pantalla de resultados
   - `07-progress.png` - Progreso por categoría
   - `08-about.png` - Acerca de

3. Actualiza el README.md con las imágenes usando:
   ```markdown
   ![Login](screenshots/01-login.png)
   ![Categorías](screenshots/04-categories.png)
   ```

## Recomendaciones

- Usa formato PNG o JPG
- Resolución recomendada: 1080x1920 o escala proporcional
- Tamaño máximo por imagen: 500KB
- Optimiza las imágenes antes de subirlas al repositorio
